# AI Mock Interview Platform

An AI-powered mock interview platform that helps students and job seekers practice interviews in a realistic environment and receive instant AI feedback.

##  Features

-  AI-generated interview questions
-  Answer recording & submission
-  AI-based feedback on responses
-  Performance insights for improvement
-  User authentication
-  Fully responsive UI
-  Deployed live on Vercel

---

## Tech Stack

- **Frontend:** Next.js (App Router) | React | Tailwind CSS  
- **Backend:** Next.js API Routes  
- **Database:** Drizzle ORM  
- **Authentication:** Clerk  
- **AI Integration:** OpenAI API  
- **Deployment:** Vercel  

---

##  How It Works

1. User signs in and selects an interview type  
2. AI generates relevant interview questions  
3. User answers the questions  
4. AI analyzes the responses  
5. Feedback is provided to help improve interview skills
